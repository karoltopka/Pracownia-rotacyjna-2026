# Pracownia Rotacyjna 2026

## Cel

Opracowanie modelu **QS\*R** (QSAR, QSTR, QSPR itp.) zgodnie z wytycznymi OECD dotyczącymi walidacji modeli.

Modele QS\*R budowane są zgodnie z **pięcioma zasadami OECD** walidacji modeli QSAR:

1. Zdefiniowany punkt końcowy (defined endpoint)
2. Jednoznaczny algorytm (unambiguous algorithm)
3. Zdefiniowana dziedzina stosowania (defined applicability domain)
4. Odpowiednia miara jakości dopasowania i predykcji (goodness-of-fit, robustness, predictivity)
5. Mechanistyczna interpretacja, jeśli możliwa (mechanistic interpretation)
